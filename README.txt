Semnal triunghiular, pozitia 16 in catalog

Cele 2 formule de baza utilizate sunt:
x(t)=suma (k=-inf, k=inf) ( C(kw)*exp(jkwt) )
C(kw)=1/P integrala(ti, ti+P) ( x(t)*exp(-jkwt)dt )

Se cunoaste ca un semnal periodic de perioada P se poate aproxima printr-o 
suma de functii sinus (sau cosinus), ale caror amplitudini sunt de fapt 
coeficientii din a doua formula, de mai sus. Semnalul reconstruit din
primii 50 de coeficienti este asemanator semnalului initial, insa exista
erori provenite in special de la calculul integralei, care nu va fi
aproximata cu o suma Riemann, ci va fi considerata o suma de arii ale unor
trapeze ce se formeaza.
